# `.createDocument()`

Creates a document.

| Argument  | Type   | Required | Description                                              |
|:----------|:-------|:---------|:---------------------------------------------------------|
| `type`    | String | Yes      | Document type                                            |
| `value`   | String | Yes      | Document value                                           |
| `options` | Object | No       | Any options you may want to pass to [`.api()`](/sdk#api) |

This method returns a **Promise**.
